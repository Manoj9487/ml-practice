# 🤖 ML Practice

Machine Learning experiments, notebooks, and small projects as I learn — covering data preprocessing, classic ML algorithms, and applied exercises on Azure ML tooling.

## Structure
```
notebooks/      → Jupyter notebooks for individual concepts/algorithms
projects/        → Small end-to-end ML projects
datasets/        → Sample datasets used (large files excluded via .gitignore)
```

## What's inside
- Data preprocessing & exploratory data analysis (pandas, numpy)
- Classic ML algorithms (regression, classification, clustering) from scratch and with scikit-learn
- Azure ML / Data Engineering exercises following Microsoft Learn modules
- Notes and takeaways from each topic

## Related Certifications
- Introduction to Machine Learning Concepts — Microsoft Learn
- Explore and Analyze Data with Python — Microsoft Learn
- Introduction to Data Engineering on Azure — Microsoft Learn
- Machine Learning — CodeChef

## Why this repo exists
Hands-on practice to complement theoretical ML coursework and certifications — building toward applied ML/data engineering roles.

```bash
pip install -r requirements.txt
```
